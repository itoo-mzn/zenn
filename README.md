# articleファイルの命名規則
- output_xxx : **公開**する記事
- draft_xxx : 公開できない状態の記事（→公開できるように編集すること）
- private_xxx : 公開しない（と決めた）記事
- memo_xxx : 個人的なメモ（→もしかしたら記事にするかもね）
- certification_xxx : 資格取得記録
